# AIカンパニー 実務版

指示ひと言で、AI役員5名(企画・開発・広報・経理・秘書)が
**事業ブリーフ / ランディングページ(HTML) / SNS告知文 / 見積書 / 業務報告書** を生成する単一HTMLアプリ。
あなたの仕事は、押印(決裁)だけ。

## 使い方

1. GitHub Pages の URL(または `index.html` をブラウザで直接)開く
2. 設定 → [platform.claude.com](https://platform.claude.com) で取得した Anthropic API キーを登録 → 接続を確認
3. 社長室で指示を入力 → 執行 → 成果物に押印 → コピー / 保存 / 共有で納品
4. 案件簿で納品・入金を記帳(入金済み合計 = 売上)

APIキーは端末のブラウザ内(localStorage)にのみ保存され、送信先は Anthropic API だけです。
キー未登録の環境ではデモモードで全フローを体験できます。
